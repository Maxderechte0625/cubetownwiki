# Plot-Befehle

Standard-Befehle für den Umgang mit deinem Plot (Alias: `/p`).

| Befehl | Beschreibung |
|---|---|
| `/plot auto` | Automatisch ein freies Plot beanspruchen |
| `/plot claim` | Das Plot beanspruchen, auf dem du gerade stehst |
| `/plot home [id]` | Zu deinem (ersten oder angegebenen) Plot teleportieren |
| `/plot visit <spieler>` | Ein fremdes Plot besuchen |
| `/plot middle` | Zur Mitte des aktuellen Plots teleportieren |
| `/plot info` | Infos zum aktuellen Plot anzeigen (Besitzer, Mitglieder, Flags) |
| `/plot list mine` | Eigene Plots auflisten |
| `/plot clear` | Plot leeren (alle Bauten entfernen, Besitz bleibt) |
| `/plot delete` | Plot löschen und wieder freigeben |
| `/plot merge <richtung>` | Angrenzendes eigenes Plot zusammenlegen |
| `/plot unlink` | Zusammengelegte Plots wieder trennen |
| `/plot swap <id>` | Zwei eigene Plots tauschen |
| `/plot rate` | Ein fertiges Plot bewerten |
| `/plot done` | Plot als "fertig gebaut" markieren |
