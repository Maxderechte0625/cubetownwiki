# Plot

Dein Plot ist dein persönliches Grundstück in der Stadt – hier darfst du frei bauen, ohne andere Spieler zu stören.

- **Größe & Lage:** je nach Rang unterschiedlich viele/größere Plots, siehe [Ränge](../../onlineshop/raenge/README.md).
- **Schutz:** Nur du (und von dir freigegebene Spieler) können auf deinem Plot bauen oder Items anfassen.
- **Beanspruchen:** Mit `/plot auto` bekommst du automatisch das nächste freie Plot zugewiesen.
