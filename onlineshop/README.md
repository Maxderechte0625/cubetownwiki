# Onlineshop

Auf [shop.cubetown.de](https://shop.cubetown.de/) kannst du deinen Support für den Server in echte Vorteile umwandeln. Der Shop ist in vier Kategorien unterteilt:

- [Ränge](raenge/README.md) – dauerhafte (bzw. Cube Rang: monatliche) Ränge mit Ingame-Vorteilen
- [Gems](gems.md) – Premium-Währung für den Gemshop
- [Bundles](bundles.md) – Rang + Gems im Kombipaket, günstiger als einzeln
- [Exclusive](exclusive.md) – einmalige Extras wie das "Verified"-Abzeichen

Käufe erfolgen per Login mit deinem Minecraft-Namen direkt im Shop.
