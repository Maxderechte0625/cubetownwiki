# Clan

Schließe dich mit anderen Spielern zusammen und kämpft Rücken an Rücken! Mit einem Clan spielt ihr als Team, sammelt gemeinsam Kills und klettert in der Clan-Bestenliste nach oben.

## Befehle

| Befehl | Beschreibung |
|---|---|
| `/clan create <name>` | Erstelle einen Clan |
| `/clan invite <player>` | Lade einen Spieler ein |
| `/clan accept` | Akzeptiere eine Clan-Einladung |
| `/clan deny` | Lehne eine Clan-Einladung ab |
| `/clan leave` | Verlasse deinen Clan |
| `/clan disband` | Löse deinen Clan auf (nur Besitzer) |
| `/clan base` | Teleportiere zur Clan-Basis |
| `/clan setbase` | Setze deine Clan-Basis |
| `/clan list [clan]` | Liste Clan-Mitglieder auf |
| `/clan stats [clan]` | Zeige Clan-Statistiken an |
| `/clan top` | Zeige die besten Clans nach Kills |
| `/clan chat` | Schalte den Clan-Chat um |
| `/clan promote <player>` | Befördere ein Clan-Mitglied |
| `/clan demote <player>` | Degradiere ein Clan-Mitglied |
| `/clan menu` | Öffne das Clan-GUI |
| `/clan members` | Öffne das Mitgliederverwaltungs-GUI |
