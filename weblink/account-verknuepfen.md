# Account verknüpfen

So verknüpfst du deinen Minecraft-Account mit Discord:

1. Verbinde dich mit `cubetown.de`.
2. Tippe im Chat `/weblink`. Du erhältst einen 8-stelligen Code.
3. Öffne [cubetown.de/link](https://cubetown.de/link) und gib den Code ein.
4. Der Code ist **10 Minuten** gültig – danach musst du `/weblink` erneut ausführen.

Nach erfolgreicher Verknüpfung landest du automatisch im [Dashboard](dashboard.md).

> Benötigt die Berechtigung `weblink.use` (standardmäßig für alle Spieler aktiv).
