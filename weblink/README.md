# WebLink (UCP)

WebLink – auch **UCP** (User Control Panel) genannt – verknüpft deinen Minecraft-Account mit deinem Discord-Account. Erreichbar unter [ucp.cubetown.de](https://ucp.cubetown.de). Damit kannst du dich einloggen und dein Inventar sowie deine Statistiken einsehen.

> WebLink und UCP bezeichnen dasselbe System – WebLink ist der interne/Plugin-Name (`/weblink`, `/webunlink`), UCP der Name der Weboberfläche.
